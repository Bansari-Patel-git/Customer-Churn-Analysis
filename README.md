**Customer Churn Analysis**

This project predicts whether a customer will churn (leave the service) using machine learning.

**What the project does**

- Cleans and prepares customer data

- Creates new features

- Trains ML models to predict churn

- Compares model performance

- Makes churn predictions for new customers

**Tools used**

- Python

- Pandas, NumPy

- Scikit-Learn

- Matplotlib

**Goal**

Help businesses identify customers who are likely to leave so they can take action to retain them.
